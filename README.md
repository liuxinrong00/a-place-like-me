# 《不完美的小店》 / A Place Like Me

《不完美的小店》是一款 2D 小店经营 / 修补模拟 / 叙事向轻策略游戏。玩家经营一家会接收旧物修补订单的小店，在有限的收入、能量、材料和真实度之间做选择：修好物品、照顾顾客，也照顾这间不完美但仍然努力存在的小店。

> 一句话卖点：在一间不完美的小店里，用有限的力气修补旧物，也慢慢修补人与生活的关系。

## 当前状态

- Unity 版本：`2022.3.62f3`
- 项目阶段：早期原型 / 文档整理
- 目标平台：PC 单机，优先 Windows，16:9 横屏，推荐分辨率 `1920x1080`
- 目标用户：18–30 岁，偏好温暖叙事、轻度模拟经营、空间布置和情绪共鸣体验的玩家
- 当前重点：先完成 MVP 核心循环，再逐步扩展顾客、口碑、升级和叙事内容。

## 设计主题

修复不是抹去破损，而是重新理解它的意义。

游戏不以“效率最大化”“收益最大化”或“完美评分”为唯一目标，而是让玩家在经营小店、修补旧物、倾听顾客故事的过程中，理解裂纹、补丁、歪斜和褪色也可以成为物品的一部分。

## 核心玩法

玩家每天从订单中选择要接的修补委托，消耗材料和能量完成修补，并在交付后获得收入、顾客反馈和口碑变化。收入用于购买材料、维护店铺和支付固定开销；能量限制每日行动数量；真实度代表小店是否仍保持自己的风格与温度。

核心循环：

1. 查看订单
2. 选择订单
3. 修补旧物
4. 交付顾客
5. 获得收入与反馈
6. 夜晚查看手机反馈 / 购买材料 / 结算
7. 开启新一天

## 主要系统

- **每日经营**：围绕订单选择、修补、交付、结算构成一天的经营节奏。
- **三核心数值**：金币、能量、真实度共同约束玩家决策。
- **订单选择**：不同订单拥有不同物品类型、损坏程度、材料需求、消耗时间、能量消耗和报酬。
- **修补方式**：完美复原、保留痕迹、创意再造带来不同收入、能耗和顾客反馈。
- **顾客反馈**：高消费且挑剔、温柔型、改造型顾客会产生不同期待。
- **口碑机制**：影响后续订单类型、客群和小店风格。
- **材料库存**：胶水、布料、木料、旧零件、颜料等材料支撑修补行为。
- **店铺升级**：工具架、材料架、展示角、休息椅、旧物展示墙带来经营能力提升。
- **手机机制**：夜晚查看店铺评价、日记和朋友圈，承载反馈与轻叙事，同时夜晚需要使用手机购买材料。
- **边界选择**：接受或拒绝加急、免费修改、压低价格、改变风格等请求。
- **长期成长**：通过每日订单积累金币与口碑，购买材料、支付开销、解锁轻量升级，并影响真实度与结局方向。

## 路线图

### MVP

- 完成一轮可玩的每日经营闭环。
- 支持基础订单列表、订单选择、材料消耗、修补结果、收入反馈和夜晚结算。
- 使用占位 UI 与简化数据结构，优先验证“选择订单 → 修补 → 结算”的乐趣。

### Alpha

- 扩展订单类型、材料类型和修补方式。
- 加入基础口碑、顾客类型和店铺升级。
- 建立基础 UI 流程：需求栏、工作台、库存、手机、结算面板。

### Beta

- 加强顾客反馈、手机机制和日记内容。
- 调整数值平衡，让收入、能量和真实度形成稳定压力。
- 补充美术风格、音效、动效和更完整的新手引导。

### 内容扩展

- 增加更多顾客故事、特殊订单、长期评价和小店变化。
- 探索多阶段剧情、更多店铺装饰和不同经营风格分支。
- 视反馈加入成就、收藏、特殊节日或主题活动。

## 开发原则

- 先做能玩的闭环，再做漂亮的表现。
- 先让选择有代价，再扩充内容数量。
- 保持“小店不完美但温柔”的核心气质。
